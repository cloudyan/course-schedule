# 中式教育课程表视觉设计系统

## 1. 设计背景与核心理念
针对中国 K-12 教育体系，本设计旨在通过色彩心理学和视觉权重，帮助学生在繁重的课程中快速定位核心学科，缓解视觉疲劳，并建立学科与色彩之间的心理关联。

### 核心原则：
- **视觉层级 (Visual Hierarchy)**：主课（语数英）拥有最高的视觉权重。
- **色彩编码 (Color Coding)**：建立学科与色彩的固定关联，形成肌肉记忆。
- **阅读舒适度**：采用低饱和度背景（浅色）搭配高对比度文字（深色），确保打印与屏幕阅读的清晰度。

## 3. 技术实现建议
1. **CSS 变量**：建议在全局 CSS 中定义这些色彩变量，以便统一维护。
2. **打印优化**：在打印模式下，确保背景色能够正常显示（使用 `print-color-adjust: exact`）。
3. **无障碍支持**：所有配色组合均经过对比度校验，确保符合 WCAG AA 级标准。
